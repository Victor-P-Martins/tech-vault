
# Disaster recovery

**RPO (Recovery Point Objective)**  -  Define um target máximo aceitável para perda de dados em um incidente

**RTO (Recovery Time Objective)** - Define um target de tempo para restauração de um sistema após uma ruptura.

**MTTR (Mean Time To Recovery)** - Métrica que representa a média de tempo real para recuperação de falhas, é uma métrica de performance histórica.
